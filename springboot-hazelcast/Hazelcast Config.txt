Its mainly used for caching the data when its first hit after that its getting from cache not from database


1) Need to add the below code in controller class
	
@CacheConfig(cacheNames = "students")
public class StudentController {

}

2) Need to annotate above the method of getStudentById
	
@GetMapping("/getStudentById/{id}")
@Cacheable(key = "#id")
public Student getStudentById(@PathVariable int id) {
	System.out.println("Getting the data from DB");
	return studentService.getStudentById(id);
}

3) Need to annotate above the method of updateStudent

@PutMapping("/updateStudent/{id}")
@CachePut(key = "#id")
public Student updateStudent(@RequestBody @Valid Student student,@PathVariable int id) {
	return studentService.updateStudent(student);
}


4) Need to annotate above the method of deleteStudentById
	
@DeleteMapping("/deleteStudentById/{id}")
@CacheEvict(key = "#id")
public String deleteStudentById(@PathVariable int id) {
	return studentService.deleteStudentById(id);
}


5) Finally need to add in main class - SpringbootHazelcastApplication

@EnableCaching
public class SpringbootHazelcastApplication {

}
